The Business Model Canvas

Key Partners

  • Course Instructor

    We receive college credit for the work placed into this project. Chris is similar to an investor or customer. One of the economic factors being sought is a grade that allows me to pass the course.

  • Code Toad Team members

    Partners are involved in creation and development of the idea into a working web application. Each member of the teams work and commitment contributes to the end goal of the project that secures a good grade and reduces our risks and uncertainty of failure.

  • GitHub and Canvas

    Acting as a place to communicate, store and share our code and product information.

Key Activities

Code Toads team members help in:

  • Organizing neighborhood watch features into user stories

  • Iterative development of user stories

  • Delegation and distribution of software work

  • Programming and configuration of neighborhood watch application

  • Tracking and managing delegation and progress of project.

Value Proposition

  • Collaboration between neighbors
